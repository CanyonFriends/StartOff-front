# StartOff

> 스터디 / 프로젝트 등 개발인원 모집을 목표로하는 웹 서비스

### 기술 스택

- 언어 : TypeScript
- 구현 : React, Redux(toolkit, saga)
- 테스트 : Jest, React-testing-library, Storybook
- 디자인 패턴 : Atomic Design Pattern
- 스타일 : emotion
- 통신 : restAPI

### 구현 목표

- ts와 아토믹 패턴의 이해를 통한 체계적인 구조
  - 가독성
  - 유지보수
  - 비효율적인 렌더링 최적화
- 테스트 커버리지 80%이상
  - 모든 컴포넌트 스토리북으로 UI테스트
  - jest, rtl으로 컴포넌트, 기타 모듈 테스트
